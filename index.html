Minutes on Air Quality Index (AQI) Project

Description: This project aims to build a neural network which is customizable by the user.
Then, it accesses the OpenWeather API for the data across the past 10 years on air quality in a specific region.
After training the model on more than 35,000 data points, it properly predicts the new information with about 93% accuracy.
